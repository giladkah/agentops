Set up Ensemble

1
Add your Anthropic API key

Ensemble uses Claude to review your PRs. Enter your API key from console.anthropic.com.

Key saved.
2
Add the GitHub Action to your repo

Copy this workflow file into .github/workflows/ensemble.yml in your repo:

{% raw %}name: Ensemble PR Review on: pull_request: types: [opened, synchronize] permissions: pull-requests: write jobs: review: runs-on: ubuntu-latest steps: - uses: giladkah/agentops/action@main with: ensemble-url: https://app.ensemblecode.dev ensemble-token: ${{ secrets.ENSEMBLE_TOKEN }} workflow: 'Bug Fix' post-results: comment{% endraw %}

Then add your ENSEMBLE_TOKEN as a GitHub repo secret (Settings → Secrets → New repository secret):
